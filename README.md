# CarePet (Vanilla)

A premium pet care application built with Vanilla HTML/CSS/JS and Python/MySQL.

## Project Structure

- `backend/`: Python HTTP Server and MySQL logic.
- `frontend/`: HTML, CSS, and JS files.

## How to Run

1.  **Database**: Import `backend/schema.sql` into MySQL.
2.  **Server**: Run `python backend/server.py` from this directory.
3.  **App**: Open `http://localhost:8000`.

## Requirements
- Python 3
- `mysql-connector-python`
- MySQL Server
